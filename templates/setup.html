First-time Setup
How to get your GitHub OAuth credentials
  1. Go to github.com/settings/developers
  2. Click New OAuth App
  3. Set Homepage URL: http://localhost:5000
  4. Set Callback URL: http://localhost:5000/auth/callback
  5. Register and copy Client ID, then generate a Client Secret

From your OAuth App page on GitHub.

Shown once when you click Generate a new client secret. Open GitHub
